# /legal-doc-drafter

1. 读取 `~/.claude/plugins/config/claude-for-legal/china-litigation-legal/CLAUDE.md` → 文书风格、引用规范、管辖法院/仲裁机构。
2. 若用户指定了案件标识，读取 `~/.claude/plugins/config/claude-for-legal/china-litigation-legal/matters/[标识]/matter.md` 获取案件事实和理论。
3. 确定文书类型并按以下对应框架起草。
4. 将审阅注记置于文书之前，列明来源、未核实事项和审阅前需完成的事项。
5. 向用户提供决策树——如何修改、如何进一步完善或直接使用。

# 法律文书起草

## 目的

起草可直接审阅和修改的中国法律文书初稿，遵循中国法律文书规范、实践档案中的文书风格，以及与案件理论一致的论证框架。

## 文书类型矩阵

| 文书类型 | 触发词 | 适用程序 |
|---|---|---|
| 起诉状 | 起诉、立案、提诉 | 民事/商事/行政一审 |
| 答辩状 | 答辩、抗辩 | 民事/商事/行政一审 |
| 上诉状 | 上诉、不服判决 | 民事/商事/行政二审 |
| 申诉状 | 申诉、再审 | 再审申请 |
| 仲裁申请书 | 仲裁、申请仲裁 | CIETAC/BAC/SHIAC等 |
| 仲裁答辩书 | 仲裁答辩、反驳申请 | 仲裁程序 |
| 代理词 | 代理词、最终陈述 | 法庭/仲裁庭陈述 |
| 辩护词 | 辩护词、刑事辩护 | 刑事诉讼辩护 |
| 律师函/催告函 | 律师函、催告、警告函 | 诉前/非诉 |
| 法律意见书 | 法律意见、法律分析 | 咨询性文书 |

## 文书起草框架

### 一、起诉状（民事/商事）

**法定要素（按《民事诉讼法》第122条）：**

```
起  诉  状

原告：[姓名/名称]
        住所/注册地：[地址]
        法定代表人/联系人：[姓名]，职务[职务]
        联系方式：[电话]

被告：[姓名/名称]
        住所/注册地：[地址]
        法定代表人/联系人：[姓名]（若已知）

诉讼请求：
一、请求判令被告[具体请求1]；
二、请求判令被告[具体请求2]；
三、本案诉讼费用由被告承担。

事实与理由：

一、基本事实

[按时间顺序陈述事实。每项重要事实后注明证据支持：（见证据X）。
事实陈述应客观、具体，避免法律评价与事实混淆。]

二、法律分析

（一）关于[争议焦点一]

[适用法律条文引用格式：《[法律名称]》第[X]条规定："[引用原文]"。
分析：[基于上述规定，本案符合/不符合……因为……]

（二）关于[争议焦点二]

[同上格式]

三、结论

综上所述，依据《[主要法律]》第[X]条，《民事诉讼法》第[X]条，特提起本诉，请求人民法院依法判决支持原告的诉讼请求。

此致
[法院名称]人民法院

证据目录：
1. 证据一：[名称]（[来源，日期]）
2. 证据二：[名称]（[来源，日期]）
……

附件：
1. [附件1]
2. [附件2]

                              原告/委托代理人：[签名]
                              [代理律师名称]律师事务所
                              联系电话：[电话]
                              [日期]
```

**引用规范：**
- 条文引用：《[法律简称]》第[X]条（第[X]款第[X]项）
- 司法解释引用：《最高人民法院关于[X]的解释》第[X]条
- 案例引用（参考而非强制）：（[年份]）[法院简称][民/商/行/刑][初/终/提]第[X]号

**常见错误检查：**
- [ ] 诉讼请求是否具体、可执行（金额、期限）？
- [ ] 事实陈述与证据是否对应？
- [ ] 法律引用格式是否规范？
- [ ] 诉讼时效是否仍在有效期内？
- [ ] 管辖法院是否正确（级别+地域）？
- [ ] 当事人信息是否完整？

### 二、答辩状

```
答  辩  状

答辩人：[姓名/名称]
        住所/注册地：[地址]
        法定代表人/联系人：[姓名]

对[原告/申请人名称]提起的[案件类型]，答辩人答辩如下：

一、答辩人认为，原告/申请人的诉讼请求不应得到支持。

二、事实与答辩理由

（一）关于[原告主张一]，答辩人认为：

[从以下角度逐一反驳：事实不符、法律不适用、证据不充分、程序有瑕疵等]

（二）关于[原告主张二]，答辩人认为：

[同上]

三、结论

综上，请求人民法院依法驳回原告全部/部分诉讼请求。

证据目录：
[同起诉状格式]

                              答辩人/委托代理人：[签名]
                              [日期]
```

### 三、代理词

```
代  理  词

尊敬的审判长/仲裁庭：

[律所名称]律师事务所接受[委托人名称]的委托，指派[律师姓名]律师担任本案[原告/被告/申请人/被申请人]的委托诉讼代理人。依据庭审调查的事实，结合相关法律规定，代理人就本案发表以下代理意见：

一、本案基本事实认定

（一）[事实1]已得到充分证明

[引用证据+法庭查明的事实，说明本事实如何支持我方主张]

二、法律适用分析

（一）[争议焦点一]

代理人认为，[核心法律论点]。

理由如下：

其一，依据《[法律]》第[X]条，[条文引用]。本案[X]符合/不符合上述规定，理由是[分析]。

其二，[第二理由]。

（二）[争议焦点二]

[同上]

三、对方意见的回应

针对[对方律师/检察官]的[主张X]，代理人认为：
[逐一反驳]

四、结论

综上所述，代理人认为[一句话结论]，请求[法院/仲裁庭][具体请求]。

代理人：[签名]
[律所名称]律师事务所
[日期]
```

```
律  师  函

[收件方全称]
收件人：[联系人姓名]（若已知）
地址：[地址]

[发件律所名称]律师事务所受[委托人名称]（以下简称"委托人"）的委托，现就[争议事项]向贵方发出如下律师函：

一、事实背景

[简述：双方关系、争议起因、已采取措施——2-3段，精简。]

二、委托人的权利依据

依据[法律条文]，委托人[有权/要求]……

三、要求

特此要求贵方在收到本函后[X]日内：
1. [具体要求1]；
2. [具体要求2]。

若贵方未在上述期限内履行，委托人将保留通过法律途径追究贵方责任的权利，包括但不限于[提起诉讼/申请仲裁/向相关部门投诉]，由此产生的一切法律后果由贵方承担。

请贵方慎重对待本函。

[律所名称]律师事务所
律师：[签名]
联系方式：[电话]
日期：[日期]

附件：[相关证明材料清单，若有]
```

### 五、辩护词（刑事）

**注意：刑事辩护词需要极度谨慎，以下为框架，必须由执业刑事律师结合具体案情和庭审情况撰写。**

```
辩  护  词

尊敬的审判长（审判员）：

[律所名称]律师事务所依法接受被告人[姓名]的委托（/被告人亲属的委托），指派[律师姓名]律师担任其辩护人。依据法庭调查已经查明的事实和相关证据，结合本案适用的法律，辩护人发表以下辩护意见：

一、[辩护观点一：无罪/罪轻/程序辩护等]

（一）[子观点]

[结合证据和法律展开论证]

二、[辩护观点二]

三、量刑意见（若认定犯罪）

若法院认定被告人[X]罪成立，辩护人认为应当从轻处理，理由如下：
1. [量刑情节一]
2. [量刑情节二]

四、结论

综上，辩护人认为[结论]，恳请法院[依法/从轻/宣告无罪]。

辩护人：[签名]
日期：[日期]
```

## 起草前检查清单

在起草任何文书之前，确认：

1. **案件理论已确认** — 是否有 `matter.md` 可读？或用户已提供案件事实？
2. **管辖机关已确认** — 法院级别和地域管辖是否正确？仲裁机构和规则已选定？
3. **诉讼时效有效** — 依据 `matter.md` 中的 `sof_deadline` 字段核实。若临近或已届满，**在文书前明显警告**。
4. **诉讼请求可执行** — 金额具体？期限具体？责任主体明确？
5. **证据清单对应** — 文书引用的证据是否已在案件文件夹中？
6. **利益冲突已完成** — 核实 `matter.md` 的冲突状态字段。

## 引用规范（强制适用）

**法律条文：**
- 格式：《[法律名称]》第[X]条（第[X]款第[X]项）
- 示例：《中华人民共和国合同法》第一百零七条（若2021年前）
- 现行：《中华人民共和国民法典》第五百七十七条（2021年1月1日起）
- 缩略：《民法典》第五百七十七条（第二次及以后引用可使用简称）

**司法解释：**
- 格式：《最高人民法院关于[X]的[解释/规定/通知]》第[X]条
- 示例：《最高人民法院关于适用〈中华人民共和国民事诉讼法〉的解释》第一百条

**案例（参考性，非强制）：**
- 中国案例仅供参考，不具有正式约束力，但最高法指导性案例例外
- 格式：（[年份]）[法院简称][庭别][字][案号]
- 示例：（2023）最高法民终123号
- 指导性案例：最高人民法院指导案例[X]号

**来源标签（严格执行）：**
- `[中国裁判文书网]` — 仅当该条文件实际出现在本次对话的检索结果中
- `[北大法宝]` / `[威科先行]` / `[国家法律法规数据库]` — 同上
- `[用户提供]` — 用户粘贴或链接了原文
- `[训练知识 — 请核实]` — 其他所有情况。这是默认值。
- `[已核实 — 最后确认日期 YYYY-MM-DD]` — 已对照权威来源核实的稳定条文引用

## 审阅注记模板

每份文书正文之前必须有审阅注记：

```
⚠️ 审阅注记
- **信息来源：** [法律数据库：已连接/未连接——引用来自训练知识，依赖前请核实]
- **已读材料：** [案件文件：matter.md + [N]份证据文件 | 用户口头提供事实 | 不适用]
- **需您确认的事项：** [N处 [审核] 标注的内联标记 | 无]
- **时效核实：** [诉讼时效截止日：YYYY-MM-DD，状态：[正常/临近/需立即处理]]
- **依赖前请做：** [实际需要做的1-2件事——若无问题则"可直接审阅修改"]

【重要提示】本文书为初稿，仅供审阅律师参考。引用的法律条文、案例和事实需在提交前独立核实。律师应结合具体案情、当事人实际情况和庭审策略对初稿进行修改。
```

## 文书质量检查（内联）

起草完成后，内联检查：

**格式：**
- [ ] 首行缩进2字符（正文段落）
- [ ] 文书标题居中、加粗
- [ ] 当事人信息完整
- [ ] 落款格式正确（签名、日期）
- [ ] 诉讼请求编号清晰（一、二、三）

**实质：**
- [ ] 每项诉讼请求均有事实和法律依据
- [ ] 法律条文引用准确（条号、款号）
- [ ] 证据与事实叙述对应
- [ ] 无自相矛盾的陈述
- [ ] 抗辩理由逻辑连贯

**风险：**
- [ ] 诉讼时效已核实
- [ ] 管辖权基础已说明（隐性或显性）
- [ ] 承认事项（不利于己方的陈述）已经主办律师确认
- [ ] 敏感信息（商业秘密等）已适当处理

## 本技能不做的事

- **代替律师做判断。** 文书框架和初稿是工具；主办律师决定策略、立场和最终措辞。
- **保证法律结果。** 每次引用均标注来源；法律适用须律师核实。
- **在没有案件事实的情况下起草。** 若无 `matter.md` 且用户未提供事实，先询问再起草。
- **承认无法看到的引用来源。** 若条文或案例来自训练知识，标注 `[训练知识 — 请核实]`，不得提升为已核实来源。
