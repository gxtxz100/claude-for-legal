# 仲裁程序管理

## 目的

仲裁是中国商事纠纷解决的重要途径，与诉讼并行。本技能覆盖仲裁程序从条款审查到文书起草的全流程管理。

## 中国仲裁法律框架

**主要法律：** 《中华人民共和国仲裁法》（1994年，2017年修订）`[训练知识 — 请核实现行版本]`

**关键原则：**
- 仲裁协议必须自愿、书面（或以书面形式确认的电子）
- 仲裁协议独立于主合同
- 一裁终局（不得上诉，只能以法定事由申请撤销）
- 仲裁机构须依法设立（非当事人自设）

## 功能一：仲裁条款分析（--check-clause）

### 仲裁条款有效性检查清单

```
仲裁条款有效性分析

条款原文：
[用户提供的条款文本]

分析结果：

1. 书面形式 ✓/✗
   [说明条款是否以书面形式存在]

2. 仲裁意图明确 ✓/✗
   [条款是否明确表达当事人将争议提交仲裁的意愿]

3. 仲裁机构/规则确定 ✓/✗
   [是否指定了特定仲裁机构或仲裁规则]
   风险：约定"到[地点]仲裁"而未指定机构，可能被认定为无效

4. 可仲裁性检查 ✓/✗
   [争议是否属于可仲裁事项：合同纠纷、财产权纠纷——排除：婚姻家庭、劳动争议须先仲裁（独立前置程序）]

5. 与其他条款冲突检查
   [是否同时约定了诉讼管辖——诉讼与仲裁同时约定时，两者均可能无效]

6. 涉外条款特殊检查（若适用）
   [是否约定了境外仲裁机构——境内当事人约定境外仲裁在某些情况下的效力]

综合结论：
[有效 | 可能无效 | 存在歧义——需法院确认 | 无效]

潜在问题：
[列出具体问题]

建议：
[修改建议或下一步]
```

### 常见仲裁条款类型

| 类型 | 有效性 | 备注 |
|---|---|---|
| "向[机构名称]仲裁委员会申请仲裁" | ✓ 有效 | 标准表述 |
| "在[城市]仲裁" | ✗ 可能无效 | 未指定机构 |
| "由仲裁委员会仲裁" | ✗ 可能无效 | 机构不明确 |
| "或诉讼或仲裁" | ✗ 无效 | 选择性条款通常无效 |
| "CIETAC仲裁" | ✓ 有效 | 机构明确 |

## 功能二：仲裁机构选择指南

### 中国主要仲裁机构

| 机构 | 简称 | 特点 | 适用案件类型 |
|---|---|---|---|
| 中国国际经济贸易仲裁委员会 | CIETAC/贸仲 | 国内最具权威的涉外仲裁机构，规则完善，可受理外国当事人 | 涉外合同、国际贸易 |
| 北京仲裁委员会 | BAC/北仲 | 北京地区商事仲裁，规则现代化，仲裁员质量高 | 商事合同、公司纠纷 |
| 上海国际仲裁中心 | SHIAC/上仲 | 上海地区，国际化程度高 | 涉外商事、金融 |
| 广州仲裁委员会 | GAC/广仲 | 华南地区 | 贸易、制造业 |
| 深圳国际仲裁院 | SCIA/深仲 | 大湾区，规则较灵活 | 科技、创新企业 |
| 中国海事仲裁委员会 | CMAC | 专门海事仲裁 | 海事、海运 |

**注：** 各仲裁机构规则、费用、程序时限均有差异，选择前应核实其最新版仲裁规则。`[训练知识 — 请核实各机构当前规则]`

### 机构选择建议框架

| 情境 | 建议机构 |
|---|---|
| 涉外合同（外国当事人）| CIETAC |
| 纯国内商事纠纷（北京）| BAC |
| 纯国内商事纠纷（上海）| SHIAC |
| 涉及中国当事人的国际仲裁 | CIETAC、SHIAC或BAC |
| 专业领域（海事）| CMAC |
| 劳动争议 | 各地劳动争议仲裁委员会（政府主导，独立体系）|

## 功能三：仲裁程序跟踪

### 典型仲裁程序时间线（以CIETAC为例）

```
仲裁程序时间线：[案件名称]

立案阶段
├── 提交仲裁申请 → [日期/待定]
├── 缴纳仲裁费 → [日期/待定]
└── 仲裁庭立案 → [日期/待定]

组庭阶段
├── 申请人提名仲裁员 → [日期/待定]（通常立案后15日内）
├── 被申请人提名仲裁员 → [日期/待定]（通常收到通知后15日内）
├── 首席仲裁员确认 → [日期/待定]
└── 仲裁庭成立 → [日期/待定]

审理阶段
├── 送达仲裁申请书及证据 → [日期/待定]
├── 被申请人答辩期限 → [日期/待定]（通常仲裁庭成立后45日内）
├── 申请人回复期限 → [日期/待定]
├── 证据交换截止 → [日期/待定]
└── 开庭审理 → [日期/待定]

裁决阶段
└── 裁决书作出 → [日期/待定]（国内仲裁通常6个月内；CIETAC国内6个月，涉外可延长）

执行阶段（若对方不自动履行）
├── 向有管辖权法院申请执行 → [日期/待定]
└── 法院强制执行程序
```

**注意：** 各机构程序时限不同，且案件复杂程度会影响实际时间。以上时限为一般参考，请以具体机构规则为准。

## 功能四：仲裁申请书框架（起草辅助）

（详细起草功能参见 `/china-litigation-legal:legal-doc-drafter 仲裁申请书`）

**仲裁申请书必要要素：**

1. 当事人信息
2. 仲裁机构和规则依据
3. 仲裁请求（具体、可执行）
4. 事实与理由
5. 证据目录
6. 仲裁员提名（如适用）

**常见错误：**
- 仲裁请求表述不具体（"判令赔偿损失"而非具体金额）
- 未计算利息至申请日
- 未附完整的仲裁协议原件或复印件
- 未缴纳仲裁费或缴费不足

## 仲裁 vs 诉讼比较（快速参考）

| 比较项目 | 仲裁 | 诉讼 |
|---|---|---|
| 前提条件 | 需有仲裁协议 | 无需协议 |
| 保密性 | 高（不公开） | 低（通常公开） |
| 终局性 | 一裁终局 | 可上诉（两审终审） |
| 执行 | 国际认可度高（《纽约公约》）| 境外执行困难 |
| 费用 | 通常较高 | 相对较低 |
| 仲裁员选择 | 当事人参与选定 | 随机分配法官 |
| 程序灵活性 | 较高 | 较低（严格法定程序） |
| 时效 | 通常较快 | 可能较长 |

## 本技能不做的事

- **确认仲裁机构的当前规则和费用。** 各机构规则持续更新，所有程序信息均标注 `[训练知识 — 请核实]` 并建议查阅机构官网最新版规则。
- **替代律师的仲裁策略判断。** 仲裁员选择、证据策略、和解时机均需律师综合判断。
- **评估境外仲裁机构（ICC/SIAC等）的程序。** 境外仲裁规则差异较大，超出本技能主要覆盖范围。
