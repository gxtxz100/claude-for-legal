# /claim-analysis

1. 读取实践档案了解风险校准和默认立场。
2. 读取用户提供的文件（律师函、诉状、或我方拟主张的事实）。
3. 按来函或发出方向分析主张。
4. 输出分析报告，附风险评级和决策树。

# 诉请/抗辩分析

## 目的

快速评估来函律师函或诉状的法律有效性（防御方），或评估我方拟提出的诉请的胜诉可能性（进攻方）。这是在决定策略之前的法律框架分析，不是最终的法律意见。

## 来函分析（--inbound）

### 分析框架

**第一步：程序性核查**
- 发函方是否有合格的代理资格？`[训练知识 — 请核实对方律师执照状态]`
- 是否明确指向特定当事人？
- 是否包含明确的法律依据和诉讼请求？
- 威胁期限是否合理？

**第二步：主张有效性分析**

按每项主张逐一分析：

```
主张 [N]：[主张简述]

法律基础：
- 引用法条：[条文，若有]
- 法条是否存在：[是/否/需核实 — 训练知识 — 请核实]
- 事实基础：[对方主张的事实]

我方应对：
- 事实层面：[我方事实是否与对方陈述一致？有何差异？]
- 法律层面：[该条文是否支持对方的主张？适用条件是否满足？]
- 证据层面：[对方的证据是否充分？我方是否有反驳证据？]

初步评估：
[强/中等/弱 — 对方此项主张的法律基础]
```

**第三步：诉讼时效核查**
- 对方主张的损害/违约何时发生？
- 是否在诉讼时效内？（若对方催告距离损害/违约发生已超过3年，可能已超时效）

**第四步：我方风险评级**

综合评估：

| 维度 | 评级 | 说明 |
|---|---|---|
| 法律有效性 | 🔴/🟠/🟡/🟢 | 对方主张的法律基础是否充分 |
| 证据风险 | 🔴/🟠/🟡/🟢 | 对方是否可能掌握关键证据 |
| 我方应对强度 | 🔴/🟠/🟡/🟢 | 我方的抗辩理由是否充分 |
| 综合风险 | 🔴/🟠/🟡/🟢 | 综合判断 |

**第五步：应对选项**

> **下一步？选择一项，我来帮您完善：**
> 1. **起草答复律师函** — 正式书面回应，承认/否认/要求补充信息
> 2. **发出反诉威胁** — 若我方存在对对方的反向主张，评估反诉/反请求可行性
> 3. **启动和解协商** — 若风险评估建议和解，起草和解意向信
> 4. **受理为正式案件** — 运行 `/china-litigation-legal:matter-intake` 建立案件档案
> 5. **仅监控不回应** — 评估：不回应是否会被视为认可或中断时效？
> 6. **其他** — 告诉我您想怎么处理

## 我方诉请分析（--outbound）

### 分析框架

**第一步：法律构成要件分析**

对于每项拟主张的权利/索赔：

```
诉请 [N]：[诉请简述]

法律依据：[适用的法律条文]
构成要件：
  1. [要件1]
     我方证据：[有/无/需补充]
     评级：🟢/🟡/🟠/🔴
  2. [要件2]
     我方证据：[有/无/需补充]
     评级：🟢/🟡/🟠/🔴
  [以此类推]

证据缺口：[哪些要件我方证据薄弱]
建议补充证据：[具体建议]
```

**第二步：诉讼时效核查**
- 损害/违约发生时间
- 时效起算点
- 时效截止日期
- 是否存在中断事由

**第三步：胜诉可能性评估**

| 诉请项目 | 法律基础 | 证据情况 | 总体评估 |
|---|---|---|---|
| [诉请1] | 强/中/弱 | 充分/可补充/缺乏 | 🟢/🟡/🟠/🔴 |
| [诉请2] | 强/中/弱 | 充分/可补充/缺乏 | 🟢/🟡/🟠/🔴 |

**第四步：诉讼请求量化**

| 项目 | 金额 | 计算依据 |
|---|---|---|
| 本金/主要损失 | [金额] | [合同条款/实际损失] |
| 利息（若主张）| [金额] | [同期LPR/约定利率] `[训练知识 — 请核实当前LPR]` |
| 违约金（若主张）| [金额] | [合同约定/实际损失的30%上限] |
| 律师费（若主张）| [金额] | [若合同约定或法律规定] |
| **合计** | **[总金额]** | |

**第五步：下一步决策树**

> **准备好起诉了吗？**
> 1. **起草起诉状** — 运行 `/china-litigation-legal:legal-doc-drafter 起诉状 [案件说明]`
> 2. **先发送催告函** — 诉前催告以中断时效并留作证据
> 3. **补充证据后再行评估** — 若证据缺口较大，列出需要收集的证据
> 4. **评估仲裁替代诉讼** — 若合同有仲裁条款，运行 `/china-litigation-legal:arbitration-management --check-clause`
> 5. **其他** — 告诉我您想怎么处理

---

## 本技能不做的事

- **给出胜诉保证。** 所有评估均为初步分析，标注 `[律师需结合完整证据材料综合判断]`。
- **替代证据审查。** 胜诉可能性评估基于用户提供的事实；完整证据材料的分析需要律师逐一审阅。
- **自动提起诉讼或仲裁。** 分析结束后提供选项；律师做决定。
