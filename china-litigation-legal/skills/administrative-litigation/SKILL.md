# 行政复议与行政诉讼

## 目的

中国行政争议解决有两条法律路径：行政复议（复议机关审查）和行政诉讼（法院审查）。两者有严格的法定期限，错过则丧失申请权利。本技能提供从分析到文书起草的全流程支持。

---

## 功能一：行政行为分析（--analysis）

### 可申请复议/诉讼的行政行为类型

```
可争议行政行为分析

用户描述的行政行为：[描述]

行为类型分类：
□ 行政处罚（罚款、吊销执照、没收等）
□ 行政许可（批准、不予批准、撤销许可）
□ 行政强制（查封、扣押、冻结账户）
□ 行政确认（行政复议决定、行政裁决）
□ 行政征收/征用
□ 行政不作为（应当作为而不作为）
□ 其他行政行为

是否可争议：[是/可能/存在障碍/否]

障碍分析（若存在）：
[例如：是否为内部管理行为？是否为"最终裁决"？]
```

### 合法性审查框架

```
行政行为合法性初步分析

1. 主体资格
   作出行政行为的机关是否有法定权限？
   [是/否/存疑 — 训练知识 — 请核实]

2. 程序合法性
   □ 是否依法告知当事人权利？
   □ 是否给予陈述、申辩机会？
   □ 是否经过法定审批程序？
   □ 是否送达合法？

3. 实体合法性
   □ 认定的事实是否有证据支持？
   □ 适用的法律是否正确？
   □ 处理结果是否适当（比例原则）？

4. 证据
   □ 我方是否有行政决定书原件？
   □ 是否记录了送达日期？（时限从此起算）

初步评估：合法性存疑 / 明显违法 / 合法但不当 / 合法
```

## 功能二：行政复议路径

### 关键时限（每次均须核实，因可能已有修订）

| 时限 | 法律规定 | 备注 |
|---|---|---|
| 申请行政复议 | **60日**内（《行政复议法》第23条）`[训练知识 — 请核实]` | 从知道或应当知道行政行为之日起算；2024年前为60日 |
| 复议机关受理审查 | 5日内 | |
| 复议决定期限 | 通常60日内（可延长30日）| |

**注意：** 2023年修订后《行政复议法》已将申请期限从60日延长至**60日**（原法规定），需查阅最新版本确认。`[训练知识 — 请核实2024年起新法下的具体期限]`

### 复议机关确定

```
行政复议机关确定

作出行政行为的机关：[机关名称]
行政行为类型：[类型]

复议机关选择：
□ 上级主管机关（通常情形）
□ 县级以上地方政府（对其工作部门行政行为）
□ 国务院（对国务院部委行政行为）

建议申请机关：[具体机关] `[审核 — 请律师确认管辖]`
```

## 功能三：行政诉讼路径

### 关键时限

| 时限 | 法律规定 | 备注 |
|---|---|---|
| 直接提起行政诉讼 | **6个月**内 | 知道或应当知道行政行为之日起算 |
| 经复议后提起行政诉讼 | 收到复议决定书**15日**内 | 若不服复议决定 |
| 不动产案件 | **20年**内 | 特殊类型 |

## 复议前置 vs 直接诉讼选择

| 情形 | 建议路径 | 原因 |
|---|---|---|
| 对处罚/许可不服 | 可选择复议或直接诉讼 | 一般情形下均可 |
| 法律规定必须先复议 | 必须先复议 | 2024年《行政复议法》扩大了复议前置范围 `[训练知识 — 请核实具体情形]` |
| 时间紧迫（接近6个月期限） | 直接诉讼 | 复议期间不影响起诉期限（除非有复议前置要求） |
| 希望调解解决 | 复议 | 复议程序有调解机制 |
| 涉及复杂法律问题 | 直接诉讼 | 法院审查更全面 |

---

## 行政赔偿

若行政行为违法且造成损失，可同时主张行政赔偿（《国家赔偿法》）：

- **人身损害：** 实际费用 + 误工损失
- **财产损害：** 直接损失（一般不含间接损失）
- **精神损害：** 造成严重后果的，可申请精神损害赔偿

## 本技能不做的事

- **保证行政复议/诉讼的结果。** 行政案件的结果受到多重因素影响；本技能提供分析框架，不预测结果。
- **评估涉及国家秘密的案件。** 若行政行为涉及国家秘密、国防等特殊领域，超出本技能范围。
- **确认最新行政法规的细节。** 行政法规变化频繁；所有引用条文均标注来源，须查阅最新版本。
